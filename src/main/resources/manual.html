Это навык Умного дома с Алисой для управления плеерами LyrionMediaServer

Установка:

  1. Должен быть запущен Lyrion Media Server https://lyrion.org

  2. Скачать архив https://github.com/knovash/squeeze-alice/blob/main/squeeze-alice-local.tar.gz

  3. Запустить скрипт install_ssh_192.168.1.123.sh скопирует файлы и запустит сервис навыка, поменяйте в названии файла на свой ip.

  4. Открыть http://192.168.1.123:8010/

  5. Нажать "Авторизация в Яндекс" - подтвердить авторизацию в Яндекс

  6. Нажать "Настройка плееров" - выбрать комнаты Умного дома для плееров LMS

  7. Открыть приложение "Дом с Алисой"

  8. Добавить Устройство умного дома

  9. Найти производителя "Lyrion Media Server"

  10. Нажать "Привязать к Яндекс"

  11. Нажать "Обновить список устройств"

  12. Показаны устройства "музыка" в комнатах

  13. Ура, теперь всё готово!

Удаление:

Запустить скрипт uninstall_ssh_192.168.1.123.sh, поменяйте в названии файла на свой ip.

Скрипт удалит файлы

rm -r /opt/squeeze-alice-1.0

rm /lib/systemd/system/squeeze-alice.service

Остановит сервис навыка

systemctl disable squeeze-alice.service

systemctl daemon-reload

В приложении "Дом с Алисой" удалите производителя и устройства
